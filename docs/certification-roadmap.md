# Certification Roadmap, Capstone Project, and 6-Month Execution Plan

Over the next six months, the target certifications are Microsoft Azure AI Fundamentals (AI-900), Azure AI Engineer Associate (AI-102), and Azure Data Scientist Associate (DP-100), acquired in that sequence and each directly tied to a performance tier: AI-900 anchors the Succeeds tier as the foundational credential, AI-102 anchors the Exceeds tier as the applied engineering credential, and DP-100 anchors the Exemplary tier as the most advanced credential covering production model development and MLOps; the capstone that encapsulates this knowledge is a secure, full-stack Azure AI application that includes retrieval-augmented generation, prompt/tool orchestration, model evaluation, telemetry, and CI/CD deployment, with months 1–2 focused on AI-900 fundamentals and baseline app setup, months 3–4 focused on AI-102 service integration and end-to-end AI workflows, and months 5–6 focused on DP-100 model lifecycle, optimization, and portfolio-ready hardening.

## 6-Month Timeline

- **Month 1:** Complete AI-900 fundamentals study path and define capstone scope/architecture.
- **Month 2:** Pass AI-900 exam and implement capstone MVP (UI, API, and core data flow).
- **Month 3:** Complete AI-102 training and integrate Azure AI services into the capstone.
- **Month 4:** Pass AI-102 exam and add production patterns (monitoring, guardrails, and evals).
- **Month 5:** Complete DP-100 training and implement model experimentation/tracking workflows.
- **Month 6:** Pass DP-100 exam, finalize performance/security hardening, and publish portfolio artifacts.

## Performance Tiers

### Succeeds — *Microsoft Certified: Azure AI Fundamentals (AI-900)*
- Complete the AI-900 training path and pass the AI-900 exam.
- Complete AI-102 and DP-100 training paths (exams not required for this tier).
- Deliver a functional capstone with documented architecture and deployment steps.
- Demonstrate baseline security controls (secrets handling, access control, input validation).

### Exceeds — *Microsoft Certified: Azure AI Engineer Associate (AI-102)*
- Pass both the AI-900 and AI-102 exams within six months.
- Deliver a production-oriented capstone with automated testing, CI/CD, and monitoring dashboards.
- Add structured evals for model quality, latency, and reliability with tracked improvement cycles.
- Produce clear technical documentation and an implementation retrospective with lessons learned.

### Exemplary — *Microsoft Certified: Azure Data Scientist Associate (DP-100)*
- Pass all three certifications (AI-900, AI-102, DP-100) on first attempt and complete ahead of the six-month target.
- Deliver an enterprise-grade capstone with cost optimization, threat modeling, incident playbooks, and governance controls.
- Implement advanced MLOps (experiment tracking, model versioning, rollback strategy, and drift monitoring).
- Publish a polished portfolio package (architecture deck, demo video, ADRs, and measurable business outcomes).
