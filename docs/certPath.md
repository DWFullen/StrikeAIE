# Azure AI Certification Path

## AI-900: Microsoft Certified: Azure AI Fundamentals
- **Exam**: AI-900
- **Certification**: [Azure AI Fundamentals](https://learn.microsoft.com/en-us/credentials/certifications/azure-ai-fundamentals/)
- **Training**: [AI-900T00-A: Introduction to AI in Azure](https://learn.microsoft.com/en-us/training/courses/ai-900t00)

## AI-102: Microsoft Certified: Azure AI Engineer Associate
- **Exam**: AI-102
- **Certification**: [Azure AI Engineer Associate](https://learn.microsoft.com/en-us/credentials/certifications/azure-ai-engineer/)
- **Training**: [AI-102T00-A: Develop AI solutions in Azure](https://learn.microsoft.com/en-us/training/courses/ai-102t00)

## DP-100: Microsoft Certified: Azure Data Scientist Associate
- **Exam**: DP-100
- **Certification**: [Azure Data Scientist Associate](https://learn.microsoft.com/en-us/credentials/certifications/azure-data-scientist/)
- **Training**: [DP-100T01-A: Designing and implementing a data science solution on Azure](https://learn.microsoft.com/en-us/training/courses/dp-100t01)

## DP-700: Microsoft Certified: Azure Data Scientist Associate (Data Engineering)
- **Exam**: DP-700
- **Certification**: [DP-700: Azure Data Scientist Associate (Note: DP-700 focuses on data engineering and analytics workloads)](https://learn.microsoft.com/en-us/credentials/certifications/azure-data-scientist/)
- **Training**: [DP-700T00-A: Designing and implementing data analytics solutions](https://learn.microsoft.com/en-us/training/courses/dp-100t01)
