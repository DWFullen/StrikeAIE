# Azure AI Certification Path

## AI-900: Microsoft Certified: Azure AI Fundamentals
- **Exam**: AI-900
- **Certification**: Azure AI Fundamentals
- **Training**: AI-900T00-A: Introduction to AI in Azure

## AI-102: Microsoft Certified: Azure AI Engineer Associate
- **Exam**: AI-102
- **Certification**: Azure AI Engineer Associate
- **Training**: AI-102T00-A: Develop AI solutions in Azure

## DP-100: Microsoft Certified: Azure Data Scientist Associate
- **Exam**: DP-100
- **Certification**: Azure Data Scientist Associate
- **Training**: DP-100T01-A: Designing and implementing a data science solution on Azure

## DP-700: Microsoft Certified: Azure Data Scientist Associate (Data Engineering)
- **Exam**: DP-700
- **Certification**: DP-700: Azure Data Scientist Associate (Note: DP-700 focuses on data engineering and analytics workloads)
- **Training**: DP-700T00-A: Designing and implementing data analytics solutions
